About the game #

This game was made by Forka137 using Godot Engine 3.

I'm a rhythm game fan and i like to type, so i wanted to create a game where i could type song lyrics. I looked through the internet and all i could find was IshoTyping. I played it for a while, but making the songs was kind of hard and it wasn't very customizable, so this looked like a good project.

Also, i'm a language learner. I've been learning for a couple of years and i think this is also a great way to keep learning words. I believe getting input and listening to songs is very good.
Sometimes you are not able to understand what you listen to. But this time since you'll be typing them, you surely will know what they said!

I hope you like the game, i will continue updating it if people show interest on it.
